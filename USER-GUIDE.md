# 📖 TamaBox 사용설명서

> **하루를 타임박스로 계획하고, 지킨 만큼 다마고치 친구를 키우는 앱.**
> 👉 https://maron2ee.github.io/timebox-quest/ (폰: 브라우저 메뉴 → "홈 화면에 추가"로 앱처럼 설치)

---

## 화면 한눈에 보기
- **상단 바(HUD)**: 레벨·경험치바, 🔥연속 달성일, ⏱️누적 실천시간, 동기화 상태(점 색)
- **탭 4개**: 🗓️ 오늘 · 📊 성과 · 🗂️ 도감 · ⚙️ 설정

---

## 🗓️ 오늘 — 계획하고 체크하기
1. **카테고리(색) 고르기** — 캐릭터 카드 아래 색 버튼(운동·주식공부 등) 중 하나를 탭.
2. **계획 칠하기** — 시간 칸을 **탭**(PC는 **드래그**로 여러 칸)하면 그 색으로 채워져요 = 계획 완성.
3. **완료 체크** — 칠한 칸을 **다시 탭**하면 칸 안 네모가 ☐ → ☑ 로 바뀌어요 = "계획 지킴!"
4. **칸 자세히 편집** — 칸 오른쪽 위 **⋯** 버튼: 카테고리 변경·메모·"실제로 한 활동"(계획과 다르게 했을 때)·비우기.
5. **도구 줄** — ◀▶ 날짜 이동, **오늘로**, **📋 어제 복사**, **⭐ 템플릿 적용/저장**(요일별 루틴 저장).
6. **오른쪽 요약** — 오늘 달성률(원형), 카테고리별 계획 vs 실천 막대.

> ⌨️ PC 단축키: 화살표=칸 이동, Enter/Space=계획·체크, Delete=비우기, E=편집, **Ctrl+Z=되돌리기**.

---

## 🐾 캐릭터 키우기 (다마고치)
- 칸을 **완료할수록 XP**가 쌓여 레벨업 → **진화**: 🥚알 → 🌱새싹이 → 🐣꼬마 → 📈주린이 → 🕶️트레이더 → 👑마스터
- 다마고치 화면 아래 **버튼 3개**: **♥ 쓰다듬기 · 🍖 밥주기 · ≡ 도감**
- 칸 완료 = **간식 🍪** 적립 → 밥을 주면 **친밀도 ❤** 상승. **포만감** 게이지는 시간이 지나면 줄어요(배고파함).
- 처음에 **고양이/강아지** 선택, 레벨이 오르면 **여러 마리 입양** 가능(도감에서 전환).

---

## 📊 성과 — 얼마나 지켰나
- **일 / 주 / 월** 전환해서 보기
- 달성률 추이, **카테고리별 시간(계획 vs 실천)**, 출석 **히트맵**
- **인사이트**(베스트 요일·최다 실천 카테고리·연속일) + **CSV 내보내기**(회고용)

---

## 🗂️ 도감 — 모으고 꾸미기
- **진화 도감**: 단계별 모습 모아보기
- **장식 아이템**: 조건 달성하면 잠금해제(모자🎀·손🎈 등) → 캐릭터에 장착
- **배경 테마**: 잠금해제한 배경으로 변경
- **내 친구들**: 새 친구 입양 / 메인 친구 전환 / 종(고양이·강아지) 변경

---

## ⚙️ 설정
- **카테고리**: 추가·편집(이름·이모지·색·하루 목표시간)
- **하루 설정**: 시작/종료 시각, 칸 단위(15/30/60분), "성공한 날" 기준 달성률, 주 시작요일, 촘촘한 보기
- **레트로 픽셀 모드** 켜기/끄기 🕹️
- **요일 템플릿 자동 적용** ⭐, **하루 알림** ⏰
- **클라우드 동기화** ☁️: 기기 간 데이터 공유(Supabase 로그인)
- **데이터 백업**: 내보내기/가져오기(JSON), 전체 초기화

---

## 💡 활용 팁 (전업투자자용)
- 장 시작 전 **리서치·주식공부** 칸을 미리 깔아두고, 장중엔 체크만.
- 카테고리별 **하루 목표시간**을 정해 "주식공부 2시간" 같은 루틴을 게임 목표로.
- 🔥 **스트릭(연속 달성) 안 끊기**를 1차 목표로 — 작은 꾸준함이 누적돼요.
- 주말에 **월간 성과**로 카테고리 균형 점검(한쪽에 치우치지 않았는지).
